API Usage Guide

API Calls:

        
            /api/v1/users           # see all users and their features
            /api/v1/user-<auth_id>/ # specific user's features

Get Json:

just add ?format=json 

        
            /api/v1/users?format=json           # see all users and their features
            /api/v1/user-<auth_id>?format=json # specific user's features
